Bir kez yapılandırın — tüm cihazlarda otomatik senkronizasyon başlar.

1

2
Sol menüden Oluşturun → Realtime Database → Veritabanı Oluşturun → bölge olarak europe-west1 seçin → Test modunda başlat
3
Sol menüden Proje Ayarları (dişli ikon) → Genel sekmesi → aşağı kaydırın → Uygulamalarınız bölümünde </> (Web) tıklayın → uygulamaya isim verin → Kaydet → açılan konfigürasyon kodundaki değerleri aşağıya yapıştırın
